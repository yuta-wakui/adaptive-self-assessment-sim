# adaptive-self-assessment-sim

This repository provides a simulation framework for **machine learning-driven adaptive self-assessment**

* 対象：WS1（１回分）／WS2（２回分）の汎用能力の自己評価データ
* 目的：質問項目を動的に提示し、質問数を削減しつつ評価制度を保つかを検証する
* 論文：〜〜（提出後にタイトル・リンクを追加）

## Installation
```bash
git clone https://github.com/yuta-wakui/adaptive-self-assessment-sim.git
cd adaptive-self-assessment-sim

# 推奨
python -m venv venv
source venv/bin/activate # Windows: venv\Scripts\activate

pit install -e .
pip install -r requirements.txt

### Run Simulation
```
